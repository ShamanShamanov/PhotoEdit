# Фотоэдит
Кнопка "Лево"
Поворачивает картинку влево

Кнопка "Право"
Поворачивает картинку направо

Кнопка "Зеркало"
Зеркалит картинку

Кнопка "Резкость"
Повышает резкость картинки

Кнопка "Ч\Б"
Делает картинку черно-белой

Кнопка "Блюр"
Блюрит картинку

Кнопка "Зашакалить"
Очень сильно повышает резкость

Кнопка "Все серое"
Превращает всю картинку в оттенки серого

Кнопка "Папка"
Выбор директории
